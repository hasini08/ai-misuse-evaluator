# AI Misuse Evaluator — Multi-Dimensional Responsible AI Assessment

This repository implements a modular Python prototype that evaluates AI use cases using a multi-dimensional responsibility framework.

The system integrates:
- **Environmental impact** (kgCO₂e emissions)
- **Ethical risk** (1–5 rubric score)
- **Creative displacement** (1–5 rubric score)
- **Purpose justification** (categorical utility)

It outputs a **Justifiability Score (0–100)** with explainable component breakdown, and includes quantitative evaluation experiments:
- sensitivity analysis
- Pareto frontier detection
- Monte Carlo robustness testing
- correlations and category breakdown

---

## Input

Place your dataset at:

`data/usecases.csv`

Required columns:
- `use_case_name`
- `emissions_kgco2e`
- `purpose_category` (`essential`, `beneficial`, `low_benefit`, `harmful`)
- `ethical_risk_score` (1–5, where 1 = lowest risk)
- `creative_displacement_score` (1–5, where 1 = lowest displacement)

Optional:
- `model_size` (metadata)

## How Scoring Works (High-Level)

Each dimension is converted into a normalised “goodness” value in [0,1]:
- Lower emissions → higher emissions component
- Lower ethical risk → higher ethics component
- Lower creative displacement → higher creativity component
- Purpose category → mapped to a utility weight

A weighted aggregation produces the final score:

`Justifiability Score = 100 * weighted_sum(components)`

Default weights:
- emissions = 0.40
- ethics = 0.25
- creativity = 0.20
- purpose = 0.15
